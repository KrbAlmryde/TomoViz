This section will walk you through the visualizer application use, specifically Volume Rendering and Slicing visualization techniques.

Open a data file

Select a new dataset file from directory strcuture by clicking on the Document-like icon.

Choose a Representation

Select a representation for instance slice, volume, etc.

Choose a Data variable

Select a data variable to visualize, for instance velocity, angular velocity, etc.

Rotate and Zoom

You can change zoom levels and rotate the object on the viewport by traditional mouse controls.

Reset Camera

If you zoom out too much and you cannot see the viewport or the object is too tiny, you can reset the Camera.
By clicking on the expanding arrow icons both in the control panel and the top bar you can reset the camera.

Now the fun Part! Choose Filters

Filter Options
From top to bottom: 1. Choose Color Preset 2. Type or autoset the scale values 3. Change opacity
Choose a Color Preset
Select a color from broad range of color spectrum.
Set Color range
You can type in a scale range by clicking on the diverging bar in the filter options. You can add value wanna specify in the textbox and then click on user icon Or you can click on Autoset icon.
Color Scaling Range Autosets
You can just click on the middle icon to do autoset which recognizes the scale range by itself.
Set Opacity
By clicking on the half light, half dark circle icon, you get a change Opacity.
Add Points
You can add points to change the opacity levels in the color scale by double clicking on the line at the poition desired.

Add Slicing to Pipeline

You can choose slice property by clicking on a 'plus' icon on the top control panel bar.

Enable Slice only

You can enable or disable Slice only by selecting or deselecting circles in the Pipeline.
Select Slice type
You can choose a Slice type.
Set Slice co-ordinates
You can enter X, Y, Z coordinate values to move the slices along any axes desired.

Update changes

To update any filter option, you should click on the little fly arrow icon.
